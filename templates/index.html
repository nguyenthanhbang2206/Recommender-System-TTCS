So sánh chi tiết 2 thuật toán deep learning

Hiệu suất thực tế của mô hình (MovieLens 1M)
Mô hình Loại dữ liệu (Feedback) Loss Function Metric đánh giá Giá trị kì vọng Tốc độ suy luận
Matrix Factorization (MF) Explicit (Rating 1-5) Mean Squared Error (MSE) RMSE, MAE RMSE ~0.88, MAE ~0.68 Cực nhanh (~0.1ms)
Neural CF (NeuMF) Implicit (Tương tác 0/1) Binary Cross-Entropy (BCE) HR@10, NDCG@10 HR@10 ~0.70, NDCG@10 ~0.42 Trung bình (~5-20ms)
Matrix Factorization
R̂[u,i] = μ + b_u + b_i + U[u]·V[i]ᵀ

• μ — global mean (≈3.5)

• b_u, b_i — user/item bias

• U[u] — user latent vector (50D)

• V[i] — item latent vector (50D)

Optimizer: SGD · Reg: L2 (λ=0.02)

Kiến trúc NeuMF (Neural Collaborative Filtering)
Output [0,1] = σ(Linear(40→1)) (Xác suất tương tác)
Concat(GMF_out[32], MLP_out[8]) → 40D Vector
GMF Branch
Tích Hadamard (32D)
emb_u(32) ⊙ emb_i(32)
MLP Branch
Các lớp ẩn + ReLU + Dropout
64 → 32 → 16 → 8
User & Item Embeddings riêng biệt (Không chia sẻ weights)
